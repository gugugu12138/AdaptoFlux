# FDIA

**一种基于功能集（包含多种类型的函数的集合）实现智能的算法**

## 项目概述
FDIA（Function Set-based Intelligence Algorithm）是一种基于功能集的智能算法。该算法通过不同类型的函数组合，实现智能计算和优化。
